# Lead2Grow Cursor Workshop — Starter

Dein Startpunkt. Was du inhaltlich baust, steht auf deiner Briefing-Karte.

## Loslegen

```bash
git clone <repo-url>
cd lead2grow-cursor-workshop
npm install
npm run dev
```

Die App läuft danach unter [http://localhost:5173](http://localhost:5173).

> **Wichtig:** Das Terminal mit `npm run dev` bleibt die ganze Session offen. Wenn du andere Befehle ausführen willst, öffne ein **zweites** Terminal-Tab.

### Terminal in Cursor öffnen

- Tastenkürzel: `Ctrl + `` (die Backtick-Taste links von der `1`)
- Menü: **View → Terminal**
- Neues Tab: `+`-Icon oben rechts im Terminal-Panel

## Fragen? Frag den Agent (Ask-Mode)

Öffne den Cursor-Chat mit `Cmd + L` (macOS) und beschreib, was du willst oder wo du hängst. Der Agent kennt das Projekt, die Ordnerstruktur, die Konventionen und alle verfügbaren Tools — du musst nichts davon auswendig wissen.

Beispiele, die gut funktionieren:

- *"Bau einen Header mit Logo links und einem Button rechts."*
- *"Wie starte ich den Dev-Server neu?"*
- *"Meine Seite ist weiß — hilf mir."*
- *"Füg eine shadcn-Select-Komponente hinzu und zeig sie in der App."*

Sei konkret. Sag *was* du sehen willst, nicht *wie* er es bauen soll.

## Schnelle Erste Hilfe


| Problem                 | Versuch zuerst                             |
| ----------------------- | ------------------------------------------ |
| Seite lädt nicht        | Läuft `npm run dev` noch? Terminal checken |
| Cursor reagiert seltsam | Cursor neu starten                         |
| Alles andere            | Frag den Agent am besten im Ask-Mode       |
